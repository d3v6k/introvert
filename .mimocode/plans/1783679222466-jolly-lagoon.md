# Plan: Intelligent Profile Handle Flow

## Current Behavior

When the profile page opens with no handle set:
1. Shows a static "FETCH ID FROM MESH" button — user must manually click
2. On click, does a single DHT lookup with 10s timeout
3. If found → populates handle, saves to profile
4. If not found → shows "No ID found" — dead end, no retry, no next step
5. Below the fetch button, a "CLAIM & VERIFY" button appears only when the user manually types a handle

## Desired Behavior

When the profile page opens with no handle set:
1. **Auto-fetch**: Immediately start fetching from mesh (no manual click needed)
2. **Loading state**: Button shows "Fetching Introvert Handle from Mesh Swarm..." with a pulsing animation
3. **Success**: Handle found → populate field, save to profile, show success
4. **Not found**: After retry exhausted → button changes to "Set Permanent Introvert Handle for This Peer ID"
5. **Error/transient failure**: Show "Fetch unsuccessful, will try again shortly" → auto-retry after 2 seconds
6. **Retry limit**: Max 3 attempts, then show the "Set Permanent Handle" button

#### 2. Add `_autoFetchHandle()` method

Called from `initState` (or `_loadProfile`) when `!_hasExistingHandle && !_isClaimed`:

```dart
void _autoFetchHandle() {
  if (_hasExistingHandle || _isClaimed || _isFetching) return;
  _fetchAttempts = 0;
  _doFetchWithRetry();
}

void _doFetchWithRetry() {
  if (_fetchAttempts >= _maxFetchAttempts) {
    setState(() {
      _isFetching = false;
      _fetchStatus = 'max_retries';
    });
    return;
  }
  _fetchAttempts++;
  setState(() {
    _isFetching = true;
    _fetchStatus = null;
  });
  _fetchIdFromMesh().then((_) {
    // If not found and not at max retries, schedule retry
    if (!_hasExistingHandle && _fetchAttempts < _maxFetchAttempts) {
      setState(() => _fetchStatus = 'retrying');
      _retryTimer = Timer(const Duration(seconds: 2), () {
        if (mounted && !_hasExistingHandle && !_isClaimed) {
          _doFetchWithRetry();
        }
      });
    }
  });
}
```

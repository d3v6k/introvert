# Comprehensive System Audit — Introvert Sovereign Messenger

**Date:** 2026-07-09
**Scope:** Full system — networking, Flutter/Dart, economy, cryptography, Solana programs

---

## Critical & High Findings

### H1: Trusted RBN Keys Contain All-Zeros Placeholders
**File:** `src/economy/daily_rewards.rs:33-38`
Two of three `TRUSTED_RBN_PUBLIC_KEYS` entries are all zeros. An attacker who could sign with a zero public key would bypass reward verification.
**Fix:** Replace with actual multisig keys or reduce array to one real key.

### H2: FileChunkRequest Amplification on RBN
**File:** `for_linux/src/network/mod.rs:4361`
O(n) database scan per unauthenticated FileChunkRequest. A malicious peer can trigger thousands of disk reads.
**Fix:** Rate-limit per peer, require contact/group membership verification.

### H3: Unique Peer IDs Leaked in Telemetry
**File:** `src/economy/mod.rs:352-356`
Telemetry envelope includes all contact PeerIDs, exposing the user's social graph to the RBN operator.
**Fix:** Hash or anonymize peer IDs before transmission.

### H4: Hardcoded Klipy API Key in Source
**File:** `lib/views/chat_features.dart:612,636`
Default API key `lh4LISPXX2w...` is embedded in the binary and shared by all users.
**Fix:** Extract to environment variable or secure config.
